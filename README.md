## Test tips
To test the char handling exercises you might want to use a file as input/output.

#### File as input
```
$ ./a.out < ~/input.txt
$ cat ~/input.txt
```

#### File as output
```
$ ./a.out >> ~/output.txt
$ cat ~/output.txt
```

#### Typing
1. EOF (Linux)
```
^D
```
2. BACKSPACE (Linux)
```
^H
```
